## Formatting Guidelines

- **Executive summary is everything:** Many leaders will only read this section. Write it last, make it count, and ensure it tells the complete story in five sentences.
- **Metrics table first, narrative second:** Lead with data, then explain. If a metric is off-track, the misses section should explain why — don't bury the explanation in the metrics table.
- **Be honest about misses:** A QBR with no misses is not credible. Every quarter has things that didn't go as planned. Frame them constructively — root cause, learning, corrective action.
- **Connect the dots:** Next quarter's priorities should clearly flow from this quarter's results. If you had a miss, the corrective action should show up in priorities. If you had a customer insight, it should inform your roadmap.
- **Resource asks need justification:** Don't just list what you need — explain why you need it and what happens without it. Make it easy for leadership to say yes.
- **Keep it presentation-ready:** This document should work as both a pre-read and a live presentation. Use tables and bullets, not long paragraphs. Target 20-30 minutes of presentation time.
- **Target length:** 4-6 pages for the core content, unlimited appendix.
